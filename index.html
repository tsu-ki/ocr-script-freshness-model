Document OCR API

Transform your document processing with our advanced OCR API. Leveraging Azure Vision and Tesseract, we offer comprehensive document analysis with high accuracy text extraction, table detection, and signature recognition.

Quick Start Guide

API Endpoints
POST /process
Content-Type: multipart/form-data

file: [PDF/Image file]
Response Format
{
    "structured_info": {
        "entities": {...},
        "dates": [...],
        "emails": [...],
        "phones": [...],
        "addresses": [...]
    },
    "tables": [...],
    "signatures": [...],
    "raw_text": "extracted text",
    "image_base64": "base64_string"
}

Code Examples

Python
import requests

files = {
    'file': open('document.pdf', 'rb')
}

response = requests.post(
    'http://your-api-endpoint/process',
    files=files
)

result = response.json()
print(f"Extracted Text: {result['raw_text']}")
print(f"Found Tables: {len(result['tables'])}")
print(f"Signatures Detected: {len(result['signatures'])}")
